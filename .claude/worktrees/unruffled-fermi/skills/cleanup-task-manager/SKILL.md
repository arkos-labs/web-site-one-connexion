# 🧹 Skill: cleanup-task-manager

Cette compétence assure que le projet reste "Clair" et facile à comprendre.

## Objectifs
1. **Maintenance de `TODO.md`** : Découper les tâches en étapes gérables.
2. **Hygiène du Code** : Supprimer les fichiers temporaires, les commentaires obsolètes et le code inutilisé.

## Instructions
- Examine régulièrement le fichier `TODO.md`.
- Si une tâche est terminée, déplace-la dans l'historique ou archive-la.
- Identifie les fichiers `.tmp`, `.gemini/` ou `scripts/_old/` et propose leur suppression.

## Actions de Clarté
- Réorganiser les dossiers trop encombrés.
- Renommer les fichiers pour qu'ils soient explicites.

> [!TIP]
> Un projet clair est un projet où un nouveau développeur peut comprendre l'architecture en moins de 5 minutes.
